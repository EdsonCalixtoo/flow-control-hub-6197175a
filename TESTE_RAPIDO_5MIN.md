# ✅ GUIA RÁPIDO DE TESTE (5 MINUTOS)

## 🚀 Comece Aqui

### PASSO 1: Abra o Console (F12)
```
Windows: F12
Mac: Cmd+Option+J

Vá em "Console" tab
```

### PASSO 2: Crie um Orçamento

```
1. Clique em "Orçamentos" no menu
2. Clique "Novo Orçamento"
3. Selecione um cliente
4. Clique "Adicionar item"
5. Selecione um produto
6. Quantidade: 5
7. Clique "Salvar Orçamento"

⏱️ Aguarde carregar
```

### PASSO 3: Observe o Console

```
Procure por:

✅ BOM:
[OrcamentosPage] ✨ SUCESSO! Orçamento PED-001 criado.
[ERP] ✅ Pedido salvo no banco com sucesso

❌ RUIM:
(nada aparece)
ou
[ERP] ❌ Erro...
```

---

## 📋 AGORA VEM O IMPORTANTE

### PASSO 4: Clique "Enviar para Financeiro"

```
No orçamento que criou:
Clique no botão "Enviar para Financeiro"
```

### PASSO 5: NO CONSOLE, Observe Atentamente

```
❓ Você vê algum desses logs?

[ERP] Status atualizado no banco: aguardando_financeiro
[ERP] ✅ Pedidos re-sincronizados

SE SIM: ✅ Envio funcionou!
SE NÃO: ❌ Algo não está acontecendo
SE VER ERRO: ❌ Erro ao enviar
```

### PASSO 6: Recarregue a Página (F5)

```
Aperte F5
```

### PASSO 7: Verifique o Status

```
✅ CORRETO:
   Orçamento ainda aparece com status "Aguardando Financeiro"

❌ ERRADO:
   Orçamento desapareceu
   
❌ MUITO ERRADO:
   Volta para formulário de criação novo
```
